# Blob Arena Multiplayer Game

## Overview

SMQ Blob Arena is a small multiplayer browser game implemented with JavaScript, HTML5 Canvas, Lua Server Pages, and the SMQ publish/subscribe protocol included with the Mako Server.

The introductory article for this game design is:

- https://makoserver.net/articles/Blob-Arena-Multiplayer-Game

## How to run

```bash
SMQ-examples/BlobArena
mako -l::www
```

SMQ RPC

Page-scoped request/response over the shared SMQ connection

This page demonstrates the custom CMS page-scoped SMQ RPC helper (REST / AJAX / RPC Over SMQ). The page uses the shared shell-level SMQ connection, calls broker-side Lua methods with scope.rpc.methodName(...), and verifies that success, failure, and concurrent calls all resolve through correlated $RpcReq / $RpcResp messages.

The standalone SMQ RPC example is documented in SMQ-examples/RPC README.md.
